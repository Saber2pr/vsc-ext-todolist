## Usage

default todo (~/.todolistrc):

- Click the `Todo StatusBar` at bottom left.
- `Ctrl+Shift+P` or `Command+Shift+P`, Execute command `todolist.main`

extra todo file support:

- Edit `*.todo`, `.todolistrc` files in the Todo editor!
